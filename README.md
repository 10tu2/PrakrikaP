# PrakrikaP

Оптовая торговля скобяными изделиями, водопроводным и отопительным оборудованием.

## Технологии

- Python 3.11+
- PyQt6
- SQLite (встроено в Python)

## Установка

```bash
pip install PyQt6
python main.py
```

## Структура проекта

| Файл | Описание |
|---|---|
| `database.py` | SQLite-слой: создание таблиц, CRUD-операции |
| `main.py` | Точка входа, главное окно PyQt6 с вкладками |
| `ui_tabs.py` | Вкладки: Товары, Заказы, Клиенты, Поставщики, Категории |
| `dialogs.py` | Диалоги добавления/редактирования для каждой сущности |

## Функциональность

- Управление товарами (название, артикул, цена, остаток, категория, поставщик)
- Управление заказами (клиент, дата, статус, сумма)
- Управление клиентами и поставщиками
- Управление категориями товаров
- Автоматический пересчёт суммы заказа по позициям
